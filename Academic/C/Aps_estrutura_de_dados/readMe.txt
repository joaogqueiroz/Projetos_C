Programas desenvolvidos para realizar a APS da materia de estrutura de dados utilizando a linguagem C.

Onde deveria ser desenvolvida uma aplicação que: 
-Transformaria uma matriz de caracteres por sua versão em numerico utilizando a tabela sugerida pelo professor.
-Realizar a cripitografia da matriz numericá utilizando uma matriz de criptografia sugerida.
- Realizar a descriptografia, onde deve ser achada a matriz inversa da matriz de criptografia, para depois multiplicarmos a inversa pela matriz criptografada, gerando a assim a matriz inicial.
